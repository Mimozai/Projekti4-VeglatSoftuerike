Projekti 4 - Menaxhimi i Kodit Burimor (Tic-Tac-Toe)

1. Përshkrimi i Projektit
Ky projekt është një implementim i lojës klasike Tic-Tac-Toe (X/O) në gjuhën C++, i zhvilluar si pjesë e lëndës "Veglat Softuerike". Qëllimi është demonstrimi i menaxhimit profesional të kodit burimor përmes GitHub, duke aplikuar kontrollin e versioneve, degëzimet (branches) dhe bashkimin e kodit përmes Pull Requests.

2. Implementimi i Logjikës (Puna ime individuale)
Në këtë projekt, unë kam qenë përgjegjëse për zhvillimin e plotë të logjikës së lojës në skedarin Source1.cpp. Funksionet e realizuara përfshijnë:
displayBoard(): Shfaqja e matricës 3x3 në konsolë.
switchPlayer(): Menaxhimi i radhës së lojtarëve 'X' dhe 'O'.
makeMove(int choice): Validimi i inputit dhe regjistrimi i lëvizjeve në tabelë.
checkWin() & checkDraw(): Algoritmet për kontrollin e fitores dhe detektimin e barazimit.

3. Menaxhimi i GitHub dhe Bashkëpunimi
Për të dokumentuar procesin e punës dhe për të përmbushur kriteret teknike, janë ndjekur këto hapa:
Branches: Zhvillimi i logjikës është bërë në degën specifike feature-logjika-mimoza.
Pull Requests (PR): Integrimi i kodit në degën main është bërë përmes Pull Request #2, duke demonstruar rishikimin dhe bashkimin e kodit.
Version Control: Përdorimi i rregullt i commits me mesazhe përshkruese për secilin ndryshim.
Konfigurimi: Përfshirja e skedarit .gitignore për të mbajtur repository-n të pastër nga skedarët e panevojshëm të Visual Studio-s.

4. Udhëzime për Setup (Nisja lokale)
Për të ekzekutuar projektin në mjedisin tuaj lokal, ndiqni këto hapa:
Clone repository-n: git clone https://github.com/Mimozai/Projekti4-VeglatSoftuerike.git
Hapni projektin: Hapni skedarin Source1.cpp në Visual Studio.
Build & Run: Compilo dhe ekzekuto kodin për të filluar lojën.
